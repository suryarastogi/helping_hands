# Product sense

## Who is this for?

Developers who want AI assistance that understands their specific codebase.
Not a generic chatbot — a repo-aware builder.

## Core value proposition

1. **Repo context**: The AI knows your file tree, naming, and conventions.
2. **Iterative building**: Conversation loop, not one-shot generation.
3. **Backend flexibility**: Choose the AI backend that fits your workflow.
4. **Persistent learning**: AGENT.md accumulates project knowledge.

## User personas

### Solo developer
- Runs CLI mode locally
- Wants fast feature iteration
- Values convention-following code

### Team lead
- Runs app mode for team
- Wants scheduled tasks (daily re-index, weekly summaries)
- Values audit trail (project log)

### CI/CD pipeline
- Runs E2E hands in CI
- Validates that the hand pipeline works end-to-end
- Values reliability and reproducibility
